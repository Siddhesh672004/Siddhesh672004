`React` `TypeScript` `Node.js` `MongoDB Atlas` `Groq Llama 3.3 70B` `Netlify OAuth` `Cloudinary` `LaTeX PDF`

- LLM-powered resume parsing → structured portfolio data → **one-click Netlify deployment with custom DNS**
- Cut manual portfolio setup time by **80%**; OAuth 2.0 + encrypted session management
- Even generates a polished LaTeX PDF resume from the same data

### 🎙️ [InterviewCoach](https://github.com/Siddhesh672004/InterviewCoach) — Adaptive AI Mock Interviewer
A voice-enabled interviewer that adapts to *your* resume, *your* target job, and how you actually answer.

`FastAPI` `LangGraph` `Groq Llama 3.3 70B` `React + Vite` `Web Speech API`

- **LangGraph state machine** drives adaptive questioning — difficulty shifts with your answers
- Full voice loop via Web Speech API; resilience and error-handling layers documented
- Multiple interview modes with structured feedback at the end

### 🛡️ [Rakshak — Safe Routes API](https://github.com/Siddhesh672004/safe-routes) — Safety-Aware Routing Engine
Finds the *safest* route, not just the fastest one.

`Python` `FastAPI` `Geospatial Analysis` `REST`

- Custom **multi-parameter safety scoring engine**: crime density, road isolation, night-time risk, distance exposure
- Serves multi-route alternatives to an Android client — production-ready architecture

### 🔐 [Auth-App](https://github.com/Siddhesh672004/Auth-App) — Full-Stack Task Manager with Hardened Auth
`Next.js 14 (App Router)` `Express` `TypeScript` `MongoDB` `JWT Access + Refresh` `Swagger` `Helmet` `Rate Limiting`

- Access + refresh token rotation with httpOnly cookies, role-based access control
- Swagger-documented API, rate limiting, bcrypt, request validation — security as a default, not an afterthought
